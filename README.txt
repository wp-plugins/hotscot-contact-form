Simple to use contact form

== Description ==

Simple contact form supporting easy form creation / customization. Saves enquiries to local site as well as sending out the usual emails.

== Installation ==

1. Extract to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Profit
